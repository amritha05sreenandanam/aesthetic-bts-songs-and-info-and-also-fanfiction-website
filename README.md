Welcome to the **BTS Universe**! This is a fan-made website dedicated to the legendary K-pop group BTS (Bangtan Sonyeondan). It serves as a portal for ARMY to explore member profiles, listen to songs, and read fan fiction.

## Features

- **Member Profiles**: Explore detailed profiles of all 7 members with aesthetic visuals.
- **Discography & Music**: Listen to curated BTS songs directly on the site.
- **Fanfic Reader**: A dedicated reader mode for enjoying fan fiction stories.
- **Aesthetic Design**: A visually pleasing "purple-themed" experience inspired by BTS colors.
- **Responsive Layout**: Works beautifully on desktop and mobile.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with animations, glassmorphism, and responsive design.
- **JavaScript**: Dynamic content loading, video player integration, and local storage management.

## Setup

1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
3. Enjoy the Borahae experience!
